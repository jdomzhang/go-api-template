# Prerequisites

## `yarn`

> npm install -g yarn

or, if you have `scoop` installed on windows

> scoop install yarn

## `fresh` - live reload

> go get -u github.com/pilu/fresh

## 'goconvey'
> go get -v github.com/smartystreets/goconvey

# Database

## `PostgreSQL`

Please check `scripts/create-database.sql` for more details

# Commands

## dev

> yarn dev

## release

> yarn release
